Local Translator Release

1. Keep the following items in the same folder:
   - translator.exe
   - _internal\
   - models\
   - workspace\
   - start.bat

2. Run start.bat.

3. The app starts a local server and opens the browser automatically.

4. If models are missing, copy these folders into models\:
   - quickmt-ja-en
   - quickmt-en-vi
   - quickmt-vi-en
   - quickmt-en-ja
